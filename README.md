## Struktura Webu
  - název Autoškola_kuba
  - téma web o autoškola
  - barevná paleta 1(#d90429) 2 (#8d99ae) 3 (#2b2d42)
  - v horní časti <head> bude menu na rychlí posun po webu a v <body> bude 
1. `<header>`
  -  popis služby
  -  telefoní číslo
  -  email
2. `<main>`
  - jaka auta nabyzím
  - cenik
  - mapu s místem kde se nachazim
  - odkaz na web s formulařem
  - data teoretických hodin
3. `<footer>`
  - konec
4. `<second page>`
  - formulář
